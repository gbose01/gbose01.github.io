PortRadar

A terminal UI that shows both host processes AND Docker container ports in one unified view

The Problem

Every developer knows this workflow: something's running on a port, you can't remember what, so you open a new terminal, type ss -tulpn (or worse, lsof -i), scroll through the output, realize it's a Docker container, then run docker ps to figure out which container owns that port.

Two separate commands. Context switching. Annoying.

I built PortRadar to fix that.

How It Works

Host Port Scanning

Instead of relying on external commands like ss or netstat (which often require elevated permissions), PortRadar reads directly from Linux's /proc filesystem:

/proc/net/tcp    # IPv4 TCP connections
/proc/net/udp    # IPv4 UDP connections  
/proc/net/tcp6   # IPv6 TCP connections
/proc/net/udp6   # IPv6 UDP connections

Each line contains the local address (IP:port in hex), connection state, and inode number. The tricky part: mapping that inode back to a process. PortRadar iterates through /proc/<pid>/fd looking for socket symlinks that match the inode.

Docker Integration

The Docker client connects to the Docker Engine API (via the Docker socket or TCP) and fetches running containers. Each container's port mappings are extracted, both published ports (host:container) and internal-only ports.

Terminal UI

Built with blessed, a ncurses-style TUI library for Node.js. It handles keyboard input, screen rendering, and interactive elements like the confirmation dialogs.

Key Features

  • Unified View: Host processes and Docker containers in one table
  • Filtering: Press 1/2/3 or 'f' to cycle: All / Host / Docker
  • Search: '/' to focus search, filter by port or process name
  • Kill Process: Select a host process, press 'k', confirm with 'y'
  • Stop Container: Select a container, press 'd', confirm with 'y'
  • Auto-refresh: Toggle with 'r', updates every 3 seconds

Tech Stack

  • Runtime: Node.js
  • TUI: Blessed
  • Docker: Dockerode
  • Platform: Linux (uses /proc)

Running It

git clone https://github.com/gbose01/portradar.git
cd portradar
npm install
npm start
